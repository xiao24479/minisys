发布介绍:

此Baidu API PHP客户端代码仅作为示范，适用于熟悉PHP但是不熟悉WebService Client开发的程序员。
本示例代码旨在为程序员提供参考和示范，各个服务的方法示例以及参数并不全面。如果需要各个服务的参数说明，
请查询完整的线上开发文档：
http://apihome.baidu.com/docs/index.php/Cn_sms_v2_sp_introduction_aboutThis/

使用过程中如果遇到任何问题，请联系apihelp@baidu.com并在标题注明《Baidu API PHP DEMO使用过程问题求助》

注意事项:

1. 如何设置SOAP Header ?
	参见soapclientcore.php。您需要填写USERNAME, PASSWORD, TOKEN。请您填写这些信息后再尝试使用此客户端。

2. 如何访问沙盒环境
	请修改soapclientcore.php中的URL。沙盒环境的URL是https://sfapitest.baidu.com

3. 本客户端的运行环境
	PHP 5.3.4 (cli) (built: Dec  9 2010 22:20:16)
